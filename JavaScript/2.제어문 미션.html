JavaScript 제어문 퀴즈 (빈칸 채우기)

다음 코드의 빈칸(____)에 들어갈 조건식 또는 제어 구문을 입력하세요.

제어문 설명

조건문

  • if : 조건이 true일 때 실행
  • if else if else : 조건을 순서대로 검사
  • switch : case가 일치하면 실행
  • 삼항 연산자 : 조건 ? 값1 : 값2

반복문

  • while : 조건이 true인 동안 반복
  • for : 초기식 조건식 증감식

반복 제어

  • break : 반복 즉시 종료
  • continue : 다음 반복으로 이동
